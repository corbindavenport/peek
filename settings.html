Peek settings

Here you can change all the options for Peek. Your changes are automatically saved.

This is what Peek uses to display Microsoft Word (.doc, docx), Excel (.xls, .xlsx), and PowerPoint (.ppt, .pptx) files. Default is Google Docs.
This changes the size of previews, in pixels. Setting a number larger than your browser window can break the preview. Default is 400.
